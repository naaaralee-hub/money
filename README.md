# 💰 공과금 관리 대시보드

공동 관리비와 공과금을 효율적으로 관리하는 웹 기반 대시보드입니다.

## ✨ 주요 기능

- 📊 **비용 관리** - 분류별 비용 입력 및 관리
- 📈 **통계 분석** - 실시간 차트 및 월별 추이 분석
- 📋 **이력 관리** - 월별 필터링된 결제 이력 조회
- 📅 **전년도 비교** - 연도별 비용 변화 추적
- 🔔 **리마인더** - 예정된 알림 설정
- 💾 **데이터 백업** - Excel, CSV, JSON 형식 지원

## 🚀 빠른 시작

### 사용 방법

1. `index.html`을 브라우저에서 열기
2. 상단의 입력 폼에서 비용 정보 입력
3. 탭을 통해 이력, 통계, 내보내기 기능 이용

### 필수 환경

- 웹 브라우저 (Chrome, Firefox, Safari, Edge 등)
- JavaScript ES6+ 지원
- 로컬 스토리지 지원

## 💾 데이터 저장

모든 데이터는 브라우저의 **로컬 스토리지**에 저장됩니다.
- 서버 필요 없음
- 개인 정보 보호
- 정기적인 백업 권장

## 🛠 기술 스택

- **프론트엔드**: HTML5, CSS3, Vanilla JavaScript
- **차트**: Chart.js v3.9.1
- **레이아웃**: CSS Grid, Flexbox
- **반응형**: 모바일, 태블릿, 데스크톱 지원

## 📊 주요 통계

- **이번 달 예상**: 현재 월의 총 비용
- **이전 달**: 지난달의 총 비용
- **평균**: 최근 6개월 평균
- **누계**: 전체 누적 비용

## 📤 데이터 내보내기

| 형식 | 설명 |
|------|------|
| **Excel** | .xlsx 형식으로 데이터 다운로드 |
| **CSV** | .csv 형식으로 데이터 다운로드 |
| **JSON** | 전체 데이터 백업 |

## 🔄 데이터 복구

백업된 JSON 파일을 통해 데이터 복구 가능
- 손쉬운 한 클릭 복구
- 기존 데이터 덮어쓰기 가능

## 📱 반응형 디자인

- 📱 **480px 이하**: 1컬럼 레이아웃
- 📱 **768px**: 2컬럼 레이아웃
- 🖥️ **1024px+**: 3-4컬럼 레이아웃
- 🖥️ **1920px+**: 최적 4컬럼 레이아웃
