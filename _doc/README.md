# 💰 공과금 관리 대시보드

공동 관리비, 공과금을 효율적으로 관리하는 웹 기반 대시보드입니다.

## 🚀 주요 기능

### 1. **비용 입력 및 관리**
- 다양한 분류별 비용 입력 (전기료, 수도료, 가스료, 인터넷, 관리비, 보험료, 기타)
- 입주자별 비용 추적
- 상세 메모 기능
- 입력한 비용 즉시 삭제 가능

### 2. **비용 분석 및 통계**
- **이번 달 예상**: 현재 월의 총 비용
- **지난달 비용**: 이전 달 총 비용
- **평균**: 최근 6개월 평균 비용
- **누계**: 전체 누적 비용
- 카테고리별 비용 분석 (원형 차트)
- 월별 비용 추이 (라인 차트)
- 월별 상세 비용 요약표

### 3. **결제 이력 관리**
- 월별로 필터링된 결제 이력 조회
- 날짜, 분류, 금액, 입주자, 메모 정보 표시
- 개별 항목 삭제
- 전체 이력 초기화 기능

### 4. **알림 및 리마인더**
- 예정된 리마인더 목록 표시
- 리마인더 추가/삭제 기능 (향후 확장 가능)

### 5. **데이터 내보내기**
- **Excel 파일로**: 현재 데이터를 Excel 형식으로 다운로드
- **CSV 파일로**: 데이터를 CSV 형식으로 다운로드
- **JSON 백업**: 전체 데이터 백업 (로컬 스토리지 포함)
- **데이터 복구**: 백업된 JSON 파일에서 복구

## 🛠 기술 스택

- **프론트엔드**: HTML5, CSS3, JavaScript (VanillaJS)
- **데이터 저장**: 브라우저 로컬 스토리지
- **차트**: Chart.js v3.9.1
- **반응형**: CSS Grid, Flexbox

## 💾 데이터 저장

모든 데이터는 **브라우저의 로컬 스토리지**에 저장됩니다.
- `expenses`: 비용 정보
- `reminders`: 리마인더 정보

**주의**: 브라우저 캐시를 삭제하면 데이터가 손실될 수 있습니다. 정기적으로 백업하세요.

## 📖 사용 방법

### 1. 비용 추가하기
1. "입력" 탭에서 비용 정보 입력
2. 분류, 금액, 날짜, 입주자 선택/입력
3. "추가하기" 버튼 클릭

### 2. 결제 이력 확인하기
1. "이력" 탭 이동
2. 월별 필터로 원하는 달의 이력 조회
3. 필요시 개별 항목 삭제

### 3. 통계 확인하기
1. "통계" 탭 이동
2. 카테고리별 비용 원형 차트 확인
3. 월별 비용 추이 라인 차트 확인
4. 월별 상세 요약표 확인

### 4. 데이터 내보내기
1. "내보내기" 탭 이동
2. 원하는 형식 선택:
   - Excel: `Excel로 내보내기`
   - CSV: `CSV로 내보내기`
   - JSON: `백업하기`
3. 파일 다운로드

### 5. 데이터 복구하기
1. "내보내기" 탭의 "데이터 복구"
2. 백업된 JSON 파일 선택
3. 확인 후 복구 진행

## 🐛 트러블슈팅

### Q: 데이터가 보이지 않습니다
A: 브라우저 개발자 도구 > Application 탭에서 로컬 스토리지 확인 후 필요하면 복구

### Q: 차트가 나타나지 않습니다
A: Chart.js 라이브러리가 로드되었는지 콘솔에서 `typeof Chart`로 확인

### Q: 파일 다운로드가 작동하지 않습니다
A: 브라우저의 팝업/다운로드 차단 설정 확인
