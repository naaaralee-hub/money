## 개요

공과금 관리 대시보드는 공동 관리비, 공과금을 효율적으로 관리하고 분석하는 웹 기반의 종합 관리 시스템입니다.

### 주요 목표
- ✅ 비용 데이터의 중앙화된 관리
- ✅ 실시간 통계 및 분석 제공
- ✅ 전년도 비교를 통한 인사이트 도출
- ✅ 데이터 보관 및 복구 기능
- ✅ 사용자 친화적인 인터페이스

**주요 특징**:
- 실시간 통계 계산
- 원형 차트 (카테고리별 비용)
- 라인 차트 (월별 추이)

- 월별 필터링
- 날짜순 정렬
- 개별 항목 삭제
- 전체 이력 초기화

**목표**: 연도별 비용 변화 추적  
**주요 특징**:
- 올해 vs 지난해 비교
- 증감량 및 증감률 표시
- 카테고리별 상세 비교
- 색상 기반 직관적 표시 (빨강/초록)

- 리마인더 추가/삭제
- 달력 기반 표시 (향후 확장 가능)

### 6️⃣ 데이터 내보내기 & 백업

## 상세 기능 설명

### 1. 비용 입력 및 관리

#### 1.1 입력 폼 구성
```
┌─────────────────────────────────┐
│ 분류: [선택 드롭다운]           │
│ 금액: [숫자 입력] ₩             │
│ 날짜: [날짜 선택]               │
│ 입주자: [텍스트 입력] (예: 101호) │
│ 메모: [텍스트 에어리어]         │
│ [추가하기 버튼]                 │
└─────────────────────────────────┘
```

#### 1.2 분류 목록
- **전기료**: 전력비
- **수도료**: 상수도요금
- **가스료**: 가스비
- **인터넷**: 통신비
- **관리비**: 공동 관리비
- **보험료**: 건물/배상 보험료
- **기타**: 기타 비용

#### 1.3 데이터 검증
- 분류: 필수 선택
- 금액: 0 이상의 숫자 (100원 단위)
- 날짜: 필수 입력
- 입주자: 필수 입력 (2글자 이상)
- 메모: 선택사항

#### 1.4 추가 기능
- **자동 날짜 설정**: 처음 로드 시 오늘 날짜 자동 입력
- **토스트 알림**: 성공/실패 메시지 표시
- **폼 초기화**: 추가 후 자동으로 폼 리셋
- **실시간 업데이트**: 추가 후 즉시 통계 갱신

### 2. 비용 분석 및 통계

#### 2.1 통계 카드 (대시보드)
```
┌──────────────────────────────────────────┐
│ 통계 항목 │ 설명                         │
├──────────────────────────────────────────┤
│ 이번 달 예상 │ 현재 달의 총 비용         │
│ 이전 달 │ 지난 달의 총 비용             │
│ 평균 │ 6개월 평균 비용                 │
│ 누계 │ 전체 누적 비용                  │
└──────────────────────────────────────────┘
```

#### 2.2 원형 차트 - 카테고리별 비용
- **표시 항목**: 각 분류별 비용 비중
- **인터랙션**: hover 시 백분율 표시
- **범례**: 하단에 카테고리 목록
- **색상**: 구별 가능한 파스텔 톤 사용

#### 2.3 라인 차트 - 월별 추이
- **기간**: 최근 12개월
- **Y축**: 비용 (₩ 포맷)
- **X축**: 월 (1월 ~ 12월)
- **포인트**: 각 월별 데이터 포인트
- **트렌드**: 곡선으로 추세 시각화

#### 2.4 요약 테이블
```
┌──────────────────────────────────────────────────┐
│ 월  │ 전기 │ 수도 │ 가스 │ 인터넷 │ 관리 │ 합계 │
├──────────────────────────────────────────────────┤
│ 2024.03 │ ... │ ... │ ... │  ...  │ ... │ ... │
│ 2024.02 │ ... │ ... │ ... │  ...  │ ... │ ... │
└──────────────────────────────────────────────────┘
```

#### 2.5 계산 로직
```javascript
// 월별 총액 = 해당 월 모든 비용의 합계
monthlyTotal = sum(expenses[year][month])

// 카테고리별 합계 = 해당 카테고리 비용의 합계
categoryTotal = sum(expenses.filter(e => e.category === cat))

// 평균 = 최근 6개월 비용 합계 / 6
average = sum(last6Months) / 6

// 누계 = 모든 비용의 합계
totalAll = sum(expenses)
```

#### 3.1 이력 조회
```
┌──────────────────────────────────────────────┐
```

- **개별 삭제**: 각 항목별 "삭제" 버튼
- **전체 삭제**: "모두 삭제" 버튼
- **확인**: 삭제 전 모달로 확인

### 4. 전년도 동월 대비 분석

#### 4.1 비교 카드
```
┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│ 올해 3월     │  │ 지난해 3월   │  │   증감량    │
│  ₩500,000   │  │  ₩450,000   │  │ +₩50,000    │
│             │  │             │  │   +11.1% ↑  │
└──────────────┘  └──────────────┘  └──────────────┘
```

#### 4.2 카테고리별 비교 테이블
```
┌────────────────────────────────────────────┐
│ 분류  │ 올해   │ 지난해 │ 증감량 │ 증감률 │
├────────────────────────────────────────────┤
│ 전기료 │ 150K  │ 130K  │ +20K  │ +15% ↑ │
│ 수도료 │ 100K  │ 120K  │ -20K  │ -17% ↓ │
│ 가스료 │ 150K  │ 150K  │  -    │  -   → │
└────────────────────────────────────────────┘
```

#### 4.3 계산 방식
```javascript
// 현재월 vs 전년도 동월
currentMonth = 3
lastYearMonth = 3
lastYear = currentYear - 1

// 증감량 = 올해 - 지난해
difference = currentTotal - lastYearTotal

// 증감률 = (증감량 / 지난해) × 100
percentageChange = (difference / lastYearTotal) × 100

// 색상 표시
if (difference > 0) → 빨강 (↑) 증가
if (difference < 0) → 초록 (↓) 감소
if (difference === 0) → 회색 (→) 동일
```

#### 4.4 카테고리별 비교
- 모든 카테고리 포함 (양쪽에 없는 것도)
- 지난해 금액이 0인 경우 백분율 = 100% (새로 추가됨)
- 개별 카테고리별 증감 추적 가능

### 6. 데이터 내보내기 & 백업

#### 6.1 Excel 내보내기
- **파일명**: `공과금_YYYY-MM-DD.xlsx`
- **포함 정보**: 날짜, 분류, 금액, 입주자, 메모
- **형식**: 표 형식, 헤더 포함
- **인코딩**: UTF-8

#### 6.2 CSV 내보내기
- **파일명**: `공과금_YYYY-MM-DD.csv`
- **구분자**: 쉼표 (,)
- **인코딩**: UTF-8
- **라인**: 헤더 포함

#### 6.3 JSON 백업
- **파일명**: `공과금_백업_YYYY-MM-DD.json`
- **포함**: 모든 비용 + 리마인더
- **버전**: 1.0
- **내용**:
```json
{
    "version": "1.0",
    "exportDate": "2024-03-27T10:30:00Z",
    "expenses": [...],
    "reminders": [...]
}
```

#### 6.4 복구 기능
- **동작**: 기존 데이터 덮어쓰기
- **확인**: 복구 전 모달 확인
- **검증**: JSON 형식 검증
- **오류**: 파일 읽기 실패 시 토스트 알림
