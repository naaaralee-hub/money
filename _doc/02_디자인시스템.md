## 개요

관리자 시스템의 기본 레이아웃은 **사이드바 + 메인 콘텐츠** 형태입니다.
- 효율적인 네비게이션
- 직관적인 사용성
- 반응형 지원

### 레이아웃 특징

```
┌─────────────────────────────────────────┐
│           헤더 (선택사항)                │
├──────────────┬──────────────────────────┤
│              │                          │
│   사이드바   │    메인 콘텐츠            │
│   (250px)    │   (풀사이즈)              │
│              │                          │
│              │                          │
├──────────────┴──────────────────────────┤
│           푸터 (선택사항)                │
└─────────────────────────────────────────┘
```

## 전체 레이아웃 구조

### 구조도

```html
<body class="admin-layout">
  <aside class="sidebar">
    <!-- 메뉴 영역 -->
  </aside>
  
  <main class="main-content">
    <!-- 콘텐츠 영역 -->
  </main>
</body>
```

### CSS Grid 기본 구조

```css
body.admin-layout {
  display: grid;
  grid-template-columns: 250px 1fr;
  grid-template-rows: auto 1fr auto;
  min-height: 100vh;
}

aside.sidebar {
  grid-column: 1;
  grid-row: 1 / -1;
}

main.main-content {
  grid-column: 2;
  grid-row: 2;
  overflow-y: auto;
}
```

### 크기 사양

| 요소 | 크기 | 설명 |
|------|------|------|
| **사이드바 너비** | 250px | 고정 너비 |
| **전체 높이** | 100vh | 뷰포트 높이 |
| **메인 콘텐츠** | 1fr | 남은 공간 |
| **패딩** | 20px | 콘텐츠 내부 여백 |

## 사이드바 메뉴

### 사이드바 구조

```
┌──────────────────┐
│  🏢 대시보드      │  ← 로고/브랜드
├──────────────────┤
│  📊 통계         │
│  💰 비용 관리    │
│  📋 이력        │
│  📤 내보내기     │
│  ⚙️  설정       │
└──────────────────┘
```

### 사이드바 스타일

**치수:**
- 너비: 250px
- 배경색: `#ffffff` 또는 `#2c3e50` (어두운 테마)
- 테두리: 1px solid `#e0e0e0`
- 스크롤: 내용 길이에 따라 자동

**컴포넌트:**

```html
<aside class="sidebar">
  <div class="sidebar-header">
    <h2>🏢 관리자</h2>
  </div>
  
  <nav class="sidebar-nav">
    <ul class="nav-list">
      <li class="nav-item">
        <a href="#" class="nav-link active">
          <span class="nav-icon">📊</span>
          <span class="nav-label">통계</span>
        </a>
      </li>
      <li class="nav-item">
        <a href="#" class="nav-link">
          <span class="nav-icon">💰</span>
          <span class="nav-label">비용 관리</span>
        </a>
      </li>
      <!-- 더 많은 메뉴 -->
    </ul>
  </nav>
  
  <div class="sidebar-footer">
    <button class="logout-btn">로그아웃</button>
  </div>
</aside>
```

### 메뉴 항목 스타일

**기본 상태:**
```css
.nav-link {
  display: flex;
  align-items: center;
  padding: 12px 20px;
  color: #666;
  text-decoration: none;
  border-left: 3px solid transparent;
  transition: all 0.3s ease;
}
```

**활성 상태:**
```css
.nav-link.active {
  background-color: #f5f7fa;
  color: #667eea;
  border-left-color: #667eea;
  font-weight: 600;
}
```

**호버 상태:**
```css
.nav-link:hover {
  background-color: #f9f9f9;
  color: #667eea;
}
```

## 메인 콘텐츠 영역

### 콘텐츠 영역 구조

```
┌────────────────────────────────────┐
│  페이지 제목 & 설명 (헤더)          │
├────────────────────────────────────┤
│                                    │
│  실제 콘텐츠 영역                   │
│  (페이지 별 다양한 컨텐츠)          │
│                                    │
│                                    │
└────────────────────────────────────┘
```

### 콘텐츠 영역 스타일

**높이 및 여백:**
```css
main.main-content {
  padding: 30px;
  overflow-y: auto;
  background-color: #f5f7fa;
}
```

**페이지 헤더:**
```html
<div class="page-header">
  <h1>페이지 제목</h1>
  <p class="page-subtitle">페이지 설명</p>
</div>
```

**페이지 헤더 스타일:**
```css
.page-header {
  margin-bottom: 30px;
}

.page-header h1 {
  font-size: 2em;
  color: #2c3e50;
  margin-bottom: 5px;
}

.page-subtitle {
  color: #7f8c8d;
  font-size: 0.95em;
}
```

### 콘텐츠 박스

**기본 카드/박스:**
```html
<div class="content-card">
  <h2>섹션 제목</h2>
  <div class="card-content">
    <!-- 실제 콘텐츠 -->
  </div>
</div>
```

**카드 스타일:**
```css
.content-card {
  background: white;
  border-radius: 8px;
  padding: 20px;
  margin-bottom: 20px;
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.08);
  transition: box-shadow 0.3s ease;
}

.content-card:hover {
  box-shadow: 0 4px 16px rgba(0, 0, 0, 0.12);
}
```

## 반응형 디자인

### 브레이크포인트

| 디바이스 | 너비 | 사이드바 | 레이아웃 |
|---------|------|---------|--------|
| 모바일 | < 768px | 숨김 (토글) | 1컬럼 |
| 태블릿 | 768px - 1024px | 축소 (150px) | 2컬럼 |
| 데스크톱 | > 1024px | 표시 (250px) | 2컬럼 |

### 모바일 반응형 CSS

```css
/* 모바일 - 사이드바 숨김 */
@media (max-width: 768px) {
  body.admin-layout {
    grid-template-columns: 1fr;
  }

  .sidebar {
    position: fixed;
    left: -250px;
    height: 100vh;
    z-index: 1000;
    transition: left 0.3s ease;
  }

  .sidebar.open {
    left: 0;
  }

  main.main-content {
    grid-column: 1;
  }

  .menu-toggle {
    display: block;
  }
}

/* 태블릿 - 사이드바 축소 */
@media (max-width: 1024px) and (min-width: 769px) {
  body.admin-layout {
    grid-template-columns: 150px 1fr;
  }

  .sidebar {
    width: 150px;
  }

  .nav-label {
    display: none;
  }

  .nav-icon {
    font-size: 1.5em;
  }
}
```

### 토글 메뉴 (모바일)

```html
<button class="menu-toggle" id="menuToggle">
  ☰ 메뉴
</button>
```

**토글 로직:**
```javascript
const menuToggle = document.getElementById('menuToggle');
const sidebar = document.querySelector('.sidebar');

menuToggle.addEventListener('click', () => {
  sidebar.classList.toggle('open');
});
```
